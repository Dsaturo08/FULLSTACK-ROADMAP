Roadmap Fullstack JavaScript en 1 Mois

Jour Contenu Projet Terminé
Semaine 1 : Les Fondamentaux (HTML, CSS, JavaScript)
Jour 1-2 HTML et CSS : Structure, Flexbox, Grid, Responsive Design CV en ligne
Jour 3-5 JavaScript de base : Variables, boucles, conditions, fonctions, DOM, événements Jeu : Devine le nombre
Jour 6-7 JavaScript avancé : ES6+, Promesses, async/await To-Do List
Semaine 2 : Front-End (React.js)
Jour 8-9 Introduction à React : Composants, JSX, props, state Compteur interactif
Jour 10-11 React avancé : Hooks (useState, useEffect), gestion des formulaires Formulaire de contact avec validation
Jour 12-13 React Router et API : Navigation, fetch/Axios Application avec API publique (ex : films)
Jour 14 Révision et projet Clone d'une application simple (ex : Netflix)
Semaine 3 : Back-End (Node.js, Express.js, Bases de Données)
Jour 15-16 Introduction à Node.js : Modules, système de fichiers, npm Serveur "Hello World"
Jour 17-18 Express.js : Création d'API RESTful, middleware, gestion des erreurs API pour liste de tâches
Jour 19-20 Bases de données : SQL (MySQL/PostgreSQL) ou NoSQL (MongoDB), ORM/ODM Connecter l'API à une base de données
Jour 21 Authentification et sécurité : JWT, bcrypt Ajouter l'authentification à l'API
Semaine 4 : Fullstack et Déploiement
Jour 22-23 Intégration Front-End et Back-End Application Fullstack (ex : blog avec authentification)
Jour 24-25 GraphQL (optionnel) Remplacer l'API REST par GraphQL
Jour 26-27 Déploiement : Vercel/Netlify (front-end), Heroku/Render (back-end) Mettre en ligne l'application Fullstack
Jour 28-30 Projet Final Plateforme de chat en temps réel avec WebSockets
Jour 31 Révision et portfolio Créer un portfolio en ligne
Jour 32 Django, Django REST, PyCharm, PostgreSQL, GitHub, Postman, CircleCI, et d'autres !
